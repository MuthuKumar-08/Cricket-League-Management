IPL Tournament Management System
📌 Overview

This project is an IPL-style Tournament Management System built using Zoho Creator (Zia AI App Builder).
The application manages teams, players, matches, automated points table updates, net run rate calculations, and playoff qualification with workflow automation and role-based access control.

The system demonstrates business application development using low-code tools with structured data modeling and automation logic.

🚀 Features
🏟 Team Management

Add and manage teams

Track matches played, wins, losses

Automatic points calculation (2 points per win)

Net Run Rate (NRR) tracking

👥 Player Management

Assign players to teams

Track runs, wickets, strike rate, economy

Role classification (Batsman, Bowler, All-rounder, Wicketkeeper)

🗓 Match Management

Schedule matches

Record match scores

Automatically calculate winner

Update team statistics after match completion

📊 Automated Points Table

Dynamic sorting based on:

Points

Net Run Rate

Auto-update after every completed match

🏆 Playoff Qualification

Automatically identifies top 4 teams after league stage

📈 Dashboard & Reports

Points table view

Top 5 run scorers

Top 5 wicket takers

Match results summary

🔐 Role-Based Access

Admin – Full control

Team Manager – Manage assigned team

Viewer – Read-only access

⚙️ Technologies Used

Zoho Creator

Zia AI App Builder

Deluge Scripting (Workflow Automation)

Role-Based Access Control

Dashboard & Report Builder

🧠 Automation Logic

When match status is marked as "Completed":

Winner is calculated automatically

Wins/Losses are updated

Points are awarded

Matches played is incremented

Net Run Rate is recalculated

Points table is dynamically sorted

Top 4 teams are selected for playoffs
